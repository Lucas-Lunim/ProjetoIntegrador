# Projeto Integrador AutoMax

Pasta final enxuta do projeto, sem subpastas e com nomes em snake_case.

## Arquivos mantidos

- `automax_300_registros_original.csv`: base original do projeto.
- `automax_300_registros_dicionario.xlsx`: dicionário de dados da base.
- `dados_tratados_registros_preenchidos.xlsx`: base tratada principal.
- `projeto_integrador_banco_corrigido_powerbi.sql`: script PostgreSQL corrigido para uso com Power BI.
- `mer_diagrama.png`: imagem do modelo entidade-relacionamento.
- `mer_modelo.mwb`: modelo editável do banco no MySQL Workbench.
- `dashboard_projeto_integrador.pbix`: dashboard do Power BI.
- `projeto_integrador_relatorio.docx`: documentação/relatório do projeto.

## Arquivos removidos da versão final

Foram removidos arquivos duplicados, backups, versões antigas, arquivos com erro de codificação, dump do banco e configurações internas do Git.
